# Tink Politie theme

v1.11.0

## What is this repository for?

The Tink Politie theme gives you all the necessary styling to develop an Politie app.

Tink is an in-house developed easy-to-use front end framework for quick prototyping and simple deployment of all kinds of websites and apps, keeping a uniform and consistent look and feel.

## Setup

### Prerequisites

* nodeJS [http://nodejs.org/download/](http://nodejs.org/download/)
* bower: `npm install -g bower`

### Install

1. Go to the root of your project and type the following command in your terminal:
   `bower install tink-theme-politie --save`

2. Include `dist/tink-politie.css` and its necessary dependencies in your project.

3. On http://tink.digipolis.be you will find all necessary documentation.
